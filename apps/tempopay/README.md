# TempoPay 💸

A payment request link generator built on the [Tempo blockchain](https://tempo.xyz).

## Live Demo
🔗 [tempopay-peach.vercel.app](https://tempopay-peach.vercel.app)

## What is TempoPay?

TempoPay lets anyone create a shareable payment request link on the Tempo network. No app download needed — just share a link and get paid.

## How it works

1. Enter your wallet address, amount, and token
2. Get a shareable link
3. Recipient opens link → connects wallet → pays in one click

## Features

- ⚡ Sub-second finality on Tempo
- 💰 Near-zero fees
- 🔗 Supports Rabby, MetaMask, WalletConnect, Coinbase Wallet
- 🪙 Supports AlphaUSD, BetaUSD, ThetaUSD, pathUSD
- 🔄 Auto network switch to Tempo Moderato (Chain ID 42431)
- 📋 Copy & share payment links instantly
